## Checks

Check your program by running the bash script `check.sh` folllowed by
the arguments necessary to execute your program. For example, if your
program were a shell script called `%ASSIGNMENT_TITLE.sh` you would
execute the check with the command `./check.sh sh
%ASSIGNMENT_TITLE.sh`.

### Check %N

#### Notes

Check %N %NOTES

#### Input

##### Standard input

Check %N %INPUT

##### Command-line arguments

Check %N %ARGS

#### Expected output

##### Standard output

Check %N %OUTPUT

##### Standard error

Check %N %ERROR

##### Exit code

Check %N %EXIT_CODE
